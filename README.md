# Sinatra Drinks Coffee
Sinatra, Thin, CoffeeScript, Haml, Compass, Susy

# Install
    $ curl -L https://get.rvm.io | bash -s stable --ruby
    $ source .rvmrc
    $ bundle install

# Start
    $ foreman start
